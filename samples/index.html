rack_resize — sample gallery

Resize to limit (contain) Scales down to fit within given dimensions, preserving aspect ratio
/samples/image_1.jpeg?width=200&height=150 /samples/image_1.jpeg?w=200&h=150

Both images are constrained to 200×150 — shorter side scales proportionally.

Resize to cover (crop) Fills the exact box by center-cropping, no empty space
/samples/image_1.jpeg?width=200&height=150&fit=cover /samples/image_1.jpeg?width=150&height=150&fit=crop
Width only / Height only Resize by a single dimension, other side scales freely
/samples/image_1.jpeg?w=180 ← width only /samples/image_1.jpeg?h=120 ← height only
Quality JPEG/WebP compression level (1–100, default 95)
/samples/image_1.jpeg?w=300&quality=95 ← q=95 /samples/image_1.jpeg?w=300&q=50 ← q=50 /samples/image_1.jpeg?w=300&q=10 ← q=10
Output formats Convert to jpeg, png, webp, avif
/samples/image_1.jpeg?w=200&format=webp /samples/image_1.jpeg?w=200&f=avif /samples/sample.png?w=200&f=jpeg
DPR (device pixel ratio) Multiplies width/height for retina screens
/samples/image_1.jpeg?w=100&dpr=1 → 100px wide /samples/image_1.jpeg?w=100&dpr=2 → 200px wide
Background color Flatten transparency when converting to JPEG; fill with a solid color
/samples/sample.png?w=200&f=jpeg&bg-color=white /samples/sample.png?w=200&f=jpeg&bg=%23a84 /samples/sample.png?w=200&f=jpeg&bg=0,128,255 /samples/sample.png?w=200&f=jpeg&background=cornflowerblue

PNG converted to JPEG with different background colors. Aliases: bg-color, background, bg.

Cloudflare URL format Alternative /cdn-cgi/image/… path format
/cdn-cgi/image/width=200,height=150/samples/image_1.jpeg /cdn-cgi/image/width=200,height=150,fit=cover/samples/image_1.jpeg /cdn-cgi/image/width=200,format=webp,quality=80/samples/image_1.jpeg
